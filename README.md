# Learn Rust: CPU Emulator

Проект по изучению языка Rust через написание низкоуровневого эмулятора процессора (архитектура, близкая к Motorola 68000).

## 🚀 О проекте
На данном этапе эмулятор умеет выполнять базовые арифметические операции, работать с подпрограммами (функциями) через стек и управлять потоком выполнения с помощью переходов.

### Основные характеристики:
- **Регистры:** 8 регистров данных (D0-D7) и 8 адресных регистров (A0-A7).
- **Память:** 4 МБ виртуальной оперативной памяти.
- **Стек:** Реализована работа со стеком (PUSH/POP) для вызова подпрограмм.
- **Цикл выполнения:** Полноценный цикл Fetch-Decode-Execute.

## 🛠 Технологии
- **Rust** (Edition 2024)
- **Зависимости:** `num`, `image`, `crossbeam` (задел на будущее развитие).

## 📥 Запуск
Для запуска эмулятора и выполнения тестовой программы:
```bash
cargo run
```
