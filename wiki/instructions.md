# Система команд эмулятора

На текущий момент процессор поддерживает следующие инструкции. Коды команд соответствуют (частично) архитектуре M68000.

## 🛠 Поддерживаемые инструкции

| Команда | Опкод (hex) | Описание |
| :--- | :--- | :--- |
| **MOVEQ** | `7xxx` | Записать 8-битное число со знаком в регистр данных. |
| **ADD** | `Dxxx` | Сложить два регистра данных, обновить флаги Z, N, C. |
| **BNE** | `66xx` | Условный переход: если флаг Z = 0, прыгнуть по смещению. |
| **JSR** | `4EB9` | Вызов подпрограммы: сохранить текущий PC в стек и прыгнуть на адрес. |
| **RTS** | `4E75` | Возврат из подпрограммы: извлечь адрес из стека и записать в PC. |

## 🔍 Детали декодирования

Декодирование происходит в методе `cpu.decode`:

1. **MOVEQ:** Маска `0x7xxx`.
   - Регистр назначения берется из битов 9-11.
   - Данные (8 бит) берутся из битов 0-7.
2. **JSR:** Уникальный код `0x4EB9`. После него ожидается 4-байтовый адрес цели.
3. **RTS:** Фиксированный код `0x4E75`.
4. **ADD:** Маска `0xDxxx` с дополнительной проверкой режима (`mode == 0x10`).
5. **BNE:** Маска `0x66xx` (где `6` — группа ветвления, а второе `6` — условие "Not Equal").
